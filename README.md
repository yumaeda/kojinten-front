# kojinten-front
Repository for Kojinten Frontend

## How to Use
### 1. Setup Mac
Refer to [mac](https://github.com/yumaeda/mac) repository.

### 2. Install required modules
```bash
npm install
```

### 3. Run code formatter & linter
```bash
npm run fix
```

### 4. Compile
```bash
npm run build
```

### 5. Run locally
```bash
npm start
```

Open the below URL:
```
http://localhost:8080
```
